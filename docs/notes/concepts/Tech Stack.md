# Tech Stack

Technologies used in Mafia Night.

## Backend

### Language
- **[[Go Language]]** - Go 1.25.4
  - Fast, compiled, type-safe
  - Excellent concurrency
  - Simple deployment

### Database
- **[[PostgreSQL]]** - PostgreSQL 16
  - Relational database
  - ACID compliance
  - JSON support

### Testing
- `testing` package (Go standard library)
- `testify` - Assertions and mocks

## Frontend

### Framework
- **[[Next.js]]** - Next.js 15 (App Router)
  - React-based
  - Server-side rendering
  - File-based routing

### Language
- **TypeScript** - Type safety for JavaScript
  - Static typing
  - Better IDE support
  - Fewer runtime errors

### Styling
- **Tailwind CSS** - Utility-first CSS
  - Rapid development
  - Consistent design
  - Small bundle size

### Testing
- **Jest** - Test runner
- **React Testing Library** - Component testing

## DevOps

### Build System
- **[[Nix Flakes]]** - Pure, reproducible builds
  - Declarative configuration
  - Automatic dependencies
  - Cross-platform

### Task Runner
- **[[Just]]** - Modern command runner
  - Simple syntax
  - Cross-platform
  - Fast

### Environment
- **[[direnv]]** - Automatic environment loading
  - Seamless activation
  - Per-directory config

### Containers
- **[[Docker Compose]]** - Multi-container orchestration
  - Development environment
  - Service dependencies

## Future Integrations

### Telegram
- **go-telegram-bot-api** - Telegram bot SDK
- Webhook-based communication

### Migrations
- **golang-migrate** - Database migrations
- Version control for schema
